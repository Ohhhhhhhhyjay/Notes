## Subfigure
```latex
\usepackage{subfigure}
\begin{figure}
\centering
\subfigure[This is a subfigure]{
    \includegraphics[width=0.45\linewidth]{Screenshot from 2022-01-17 22-39-24.png}
}
\subfigure[This is a subfigure]{
    \includegraphics[width=0.453\linewidth]{index2.png}
}
\caption{This is a figure with two subfigures}
\label{fig:star}
\end{figure}
```
